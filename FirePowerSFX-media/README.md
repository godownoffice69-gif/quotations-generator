# FirePowerSFX Media Library

This folder contains organized photos and videos showcasing our pyrotechnics work across 25 years and 5000+ events.

## Folder Organization

### Event Type Categories

- **wedding-entries/** - Grand wedding entrance displays with special effects
- **reception/** - Reception pyrotechnics including dance floor effects, cake cutting displays
- **corporate/** - Corporate events, product launches, brand activations
- **aerial-shots/** - Outdoor aerial fireworks displays
- **synchronized/** - Music-synchronized fireworks shows
- **machines/** - Cold pyro machines, sparklers, CO2 jets, confetti cannons

### Supporting Content

- **testimonials/** - Customer photos, video testimonials, thank you messages
- **raw-unorganized/** - Temporary folder for new uploads before categorization

## File Naming Convention

Use descriptive names that help customers understand what they're seeing:

```
event-type_effect-name_venue-type_YYYYMM.jpg
```

**Examples:**
- `wedding-entry_heart-sparklers_palace_202401.jpg`
- `reception_dance-floor-cold-pyro_banquet-hall_202312.jpg`
- `corporate_product-launch_convention-center_202403.jpg`

## Image Guidelines

- **Format:** JPG for photos, MP4 for videos
- **Size:** Optimize for web (max 500KB for photos, compress videos)
- **Resolution:** 1920x1080 for landscape, 1080x1920 for vertical
- **Quality:** Show clear visuals of effects, venue setup, customer reactions

## Gallery Display

These images will be displayed on the quotation website with:
- Filtering by event type
- Lightbox for full-screen viewing
- Mobile-optimized responsive gallery
- "See this effect in your event" call-to-action buttons
